# 有鹿机器人客户帮助中心项目交接说明

## 一、项目名称

有鹿机器人客户帮助中心 / 售后知识库系统

当前项目目录：

youlu-help-center-clean

## 二、项目技术栈

- Next.js 15.5.19
- React 19
- TypeScript
- Tailwind CSS
- Prisma 6.19.0
- SQLite 本地数据库
- bcryptjs 密码加密

## 三、当前已完成功能

### 1. 客户端功能

- 首页根据登录身份显示不同入口
- 客户登录
- 客户提交售后工单
- 客户只能查看自己账号下的工单
- 客户可查看教程视频
- 未登录不能直接进入客户工单页面

### 2. 管理员后台功能

- 管理员登录
- 后台工单管理
- 查看全部客户工单
- 修改工单状态：待处理 / 处理中 / 已解决
- 查看工单处理记录
- 用户管理
- 新增客户账号
- 启用 / 禁用客户账号
- 视频管理
- 新增视频记录
- 删除视频记录
- 播放视频
- 上传 mp4 视频文件
- 后台新增视频后，客户教程页自动展示

### 3. 权限功能

- 管理员可以进入后台
- 客户不能进入后台
- 未登录用户会跳转登录页
- 被禁用账号不能登录
- 密码已使用 bcryptjs 加密存储

## 四、默认账号

管理员账号：

admin / 123456

客户测试账号：

customer / 123456

如果数据库被恢复或重置，可以运行：

### 客户端

- 首页：http://localhost:3000

- 工单管理：http://localhost:3000/admin/tickets
- 用户管理：http://localhost:3000/admin/users
- 视频管理：http://localhost:3000/admin/videos

## 六、主要 API

### 登录

POST /api/login

### 工单

GET /api/tickets  
POST /api/tickets  
PATCH /api/tickets/status  
GET /api/tickets/logs?id=工单ID

### 用户

GET /api/users  
POST /api/users  
PATCH /api/users

### 视频

GET /api/videos  
POST /api/videos  
DELETE /api/videos?id=视频ID  
POST /api/videos/upload

## 七、数据库说明

数据库类型：SQLite

数据库文件：

prisma/dev.db

数据库结构文件：

prisma/schema.prisma

当前主要数据表：

- User：用户表
- Ticket：工单主表
- TicketLog：工单处理记录表
- Video：视频表

如果修改数据库结构，需要运行：

npx prisma db push
npx prisma generate

## 八、本地启动方式

进入项目目录：

cd youlu-help-center-clean

安装依赖：

npm install

生成 Prisma Client：

npx prisma generate

同步数据库：

npx prisma db push

启动项目：

npm run dev

打开：

http://localhost:3000

## 九、重要提醒

1. 不要上传 node_modules 和 .next，这两个可以重新生成。
2. 当前数据库是 SQLite，只适合本地开发和演示。
3. 正式上线建议改为 PostgreSQL 或 MySQL。
4. 当前视频上传到 public/uploads/videos，本地可用。
5. 正式上线建议改为阿里云 OSS / 腾讯云 COS / AWS S3。
6. 当前登录状态使用 localStorage，正式上线建议改为 Cookie / Session / JWT。
7. 当前上传大视频可能占用本地磁盘，正式版本需要增加文件大小限制和云存储。
8. 最近稳定备份目录为：youlu-help-center-clean-v14-video-tutorial-db-stable。
9. 当前开发目录为：youlu-help-center-clean。
